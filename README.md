# SiKnight | Interactive Gaming Portal

**High-Engagement Community Hub & Immersive Interaction Engine.**

SiKnight is a high-fidelity gaming nexus engineered for community engagement. It synthesizes a medieval fantasy aesthetic with modern gaming telemetry, featuring a custom HTML5 Canvas engine and dynamic state-managed leaderboards.

## ⚔️ System Architecture
- **Coin Knight Engine:** A custom-built HTML5 Canvas game with physics-based kinematics and real-time score tracking.
- **Dynamic Leaderboard Protocol:** State-managed scoring system with category-specific data filtering.
- **Kinetic UI/UX:** Intersection Observer API-driven animations and hardware-accelerated SVG kinematics.

## 🛠️ Technical Stack
- **Vanilla JavaScript (ES6+):** Custom DOM orchestration and centralized state governance.
- **Performance CSS3:** Sophisticated visual effects including high-fidelity glows and scanline overlays.
- **Modular Asset Pipeline:** Hand-coded SVG architecture for optimized load times and fluid responsivity.
